Donations

PDFBox is free software, if have found it useful or has saved you money over a commercial alternative then consider making a donation to the project. Donations help keep the project active and support development of new features.

There are many ways that you can donate to the PDFBox project, below are some suggested ways:

Donation Description
ASN Membership The Adobe Solutions Network allows for insider information to Adobe products, which can help implement new features in PDFBox. This will also allow PDFBox be listed as a partner on Adobe's site which will increase the user community of PDFBox. Please contact Ben to make arrangements.

Note:The premium membership includes Acrobat 7.0 Professional, which is the next item on the list!
Adobe® Acrobat 7.0 Professional In order for PDFBox to produce correct results it is sometimes necessary to create a PDF using an Adobe tool and then compare the results with what PDFBox produced. The 7.0 product suite contains lots of useful features that would be nice to be able to verify against PDFBox. Amazon currently lists this as $409, but may be cheaper else where. Please contact Ben to make arrangements.(Legal donations only please)

Note:Many open issues are related to AcroForms, this donation will make it easier to implement a robust solution!
SourceForge subscription An annual SF subscription is currently $40 and provides some niceties for PDFBox developers. Please contact Ben to make arrangements. Covered for 2005/2006!!
Monetary Monetary donations are also gladly accepted. There are two ways to donate to the PDFBox project:
Donate via SourceForge
Donate directly
Cash donations will be used for
  • Project expenses such as domain registration and web hosting
  • Perks for the developers to keep them motivated and fed
  • Marketing materials such as brochures/stickers/posters
  • Expenses for participation at PDF related trade shows and conferences
